# Запити зацікавлених осіб

## Вступ

Цей документ створений для опису змісту, роботи та функціональності системи аналізу медіа контенту, а також для формування запиту зацікавлених осіб.

### Мета 

Метою є створення інформаційної системи, що дозволяє збирати різні метрики медіа контенту та на основі них створювати статистичні дані або аналітичний звіт.

### Контекст

Даний документ пов’язаний з проектами, які за допомогою технології Big Data накопичують великий обсяг інформації, аналізують його і створюють кінцевий продукт у вигляді інформативного звіту.


### Основні визначення та скорочення

Інтернет-ЗМІ (Інтернет-медіа) -  інформаційний сайт, який регулярно оновлюється і виконує функцію засобу масової інформації (ЗМІ), користується певною популярністю і авторитетом (має свою постійну аудиторію).  
PR - зв'язки з громадськістю, «піар».  
Медіахолдинг - це корпорація, яка займається збором інформації, а також її тиражуванням. Найчастіше медіакомпанії пов'язують себе з телебаченням, рекламою.  
Big Data (Великі Дані) - набори інформації (як структурованої, так і неструктурованої) настільки великих розмірів, що традиційні способи та підходи (здебільшого засновані на рішеннях класу бізнесової аналітики та системах управління базами даних) не можуть бути застосовані до них.  
Соціальна мережа – це мережа людей, які зустрічаються в Інтернеті для спілкування, розміщуючи інформацію та зображення, залишаючи коментарі чи надсилаючи повідомлення.  
Machine Learning (машинне навчання) - підгалузь штучного інтелекту в галузі інформатики, яка часто застосовує статистичні прийоми для надання комп'ютерам здатності «навчатися» (тобто, поступово покращувати продуктивність у певній задачі) з даних, без того, щоби бути програмованими явно.  
Контент - вміст, інформаційне наповнення засобу масової комунікації.
